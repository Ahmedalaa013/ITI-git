# Ahmed Alaa Ibrahim Git lab 2

### Deleting a branch LOCALLY

git branch -d <branch>

### Removing branch from remote

git push -d origin <branchname>
git branch -d <branchname>

### To list Tags

git tag

### Delete remote tag

git push origin --delete <tagname>

### Delete local tag

git tag -d <tagname>
